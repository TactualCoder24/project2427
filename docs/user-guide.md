# Vidvas AI - User Guide

## Getting Started

### 1. Create Your Account
1. Visit https://your-domain.com
2. Click "Sign Up"
3. Enter your email and password
4. Verify your email

### 2. Connect Your Apps
1. Go to **Integration Hub**
2. Click "Connect" on desired apps
3. Authorize access
4. Apps will show as "Connected"

### 3. Create AI Personas
1. Navigate to **Personas**
2. Click "Create New Persona"
3. Choose an icon and name
4. Write instructions for the AI
5. Select connected apps
6. Save your persona

### 4. Build Workflows
1. Go to **Workflow Builder**
2. Click "Create New Workflow"
3. Choose a trigger (Manual, Scheduled, Webhook)
4. Add workflow steps
5. Activate the workflow

### 5. Monitor Executions
1. Visit **Execution Dashboard**
2. View all workflow runs
3. Click on any execution for details
4. Check success rates and errors

## Troubleshooting

**Q: Integration won't connect**
A: Make sure you've authorized all requested permissions

**Q: Workflow not running**
A: Check that the workflow is "Active" and trigger is configured

**Q: Can't see my data**
A: Refresh the page or check your internet connection
